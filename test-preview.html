CleanPreview 独立预览测试

1. 通过本地文件访问

直接打开构建后的文件：

打开预览页面
注意：需要在浏览器中允许本地文件访问

2. 通过本地服务器访问

启动本地服务器后访问：

npx serve dist/preview-app -p 3000
本地服务器预览

3. 通过URL参数传递页面ID

使用 ?id=页面ID 参数：

http://localhost:3000/preview-app.html?id=YOUR_PAGE_ID
测试页面ID

4. 通过URL参数直接传递页面数据

使用 ?data=BASE64_ENCODED_DATA 参数：

http://localhost:3000/preview-app.html?data=eyJpZCI6InRlc3QiLCJuYW1lIjoiVGVzdCBQYWdlIn0=
数据需要是Base64编码的JSON格式

5. 通过全局变量传递数据

在页面中设置全局变量：

<script> window.pageData = { id: "your-page-id", name: "页面名称", root: { /* 页面结构数据 */ } }; </script>

6. 通过localStorage传递数据

在控制台设置：

const pages = [{id: "test", name: "测试页面", root: {/*...*/}}]; localStorage.setItem('studio.pages', JSON.stringify(pages));

构建命令

npm run build:preview-app

构建输出目录：dist/preview-app/

测试数据示例

使用以下JavaScript在控制台添加测试数据：

const testPage = { id: "d1cfee40-aa8e-4398-9561-ed734a8a4141", name: "测试页面", root: { id: "root", type: "div", props: { className: "p-4" }, children: [ { type: "h1", props: { children: "Hello World" } }, { type: "p", props: { children: "这是一个测试页面" } } ] } }; localStorage.setItem('studio.pages', JSON.stringify([testPage]));
